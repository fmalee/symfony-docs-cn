这些简短的文章解释了为Symfony文档做出贡献所需的一切：

:doc:`贡献流程 </contributing/documentation/overview>`
  解释了贡献修复和新内容的步骤。这是大多数开源项目所遵循的相同的贡献过程，因此你可能已经知道所需的一切。

:doc:`文档格式 </contributing/documentation/format>`
  解释用于编写文档的reStructuredText格式的技术细节。如果你已熟悉此格式，请跳过它。

:doc:`文档标准 </contributing/documentation/standards>`
  解释如何编写文档和代码示例以匹配现有文档的其余部分的风格和基调。

:doc:`许可证 </contributing/documentation/license>`
  解释用于Symfony文档的Creative Commons BY-SA 3.0许可证的详细信息。
