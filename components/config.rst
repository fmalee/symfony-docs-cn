Config组件
====================

    Config组件提供了几个类来帮助你查找、加载、组合、自动填充和验证任何类型的配置值，
    无论它们的来源是什么（YAML、XML、INI文件，或一个数据库）。

安装
------------

.. code-block:: terminal

    $ composer require symfony/config
